1. So sánh R với Python cơ bản#

Để có thể hỗ trợ dễ dàng hơn nhóm phân tích dịch chuyển từ R, chương này tập trung so sánh R & Python trên 3 nhóm tác vụ

  • Các câu lệnh cơ bản

  • Biến đổi và xử lý dữ liệu

  • Trực quan hóa dữ liệu

Với R, các câu lệnh sẽ được sử dụng với hệ sinh thái của tidyverse. Với python, các câu lệnh sẽ được sử dụng chủ yếu với pandas, numpy, os và plotnine

1.2.2. Nhóm biến đổi dữ liệu#

STT

Hàm

Python

R

1

Đổi tên

df.columns = ["P", "Q"]

names(df) = c("P", "Q")

df.rename(columns = {old_var : new_var})

df %>% rename(new_var = old_var)

2

Lấy biến a & b

df.filter(['a', 'b'])
df[['a', 'b']]

df %>% select(a, b)

3

Lọc theo dòng từ 1-4

df.iloc[0:4]

df %>% slice(1:4)

4

Lọc theo điều kiện

df[(df.a>20) & (df.b ==10)]

df %>% filter (a > 20 & b == 1)

df.query('a > 20 and b == 1')

5

Sắp xếp lại dữ liệu

df.sort_values('a')

df %>% arrange(a)

6

Loại bỏ 1 biến

df.drop('a', axis = 'columns')

df %>% select(-a)

7

Tạo biến mới theo nhóm

df['new_var'] = df.groupby('a')['x'].transform('mean')

df %>%
group_by(a) %>%
mutate(new_var = mean(x))

8

Left join

pd.merge(df1, df2, by = "left', left_on = 'id1', right_on = 'id2')

df1 %>% left_join(df2, by = c('id1' = 'id2'))

9

Ghép dòng

pd.concat(df1, df2)

bind_rows(df1, df2)

10

Ghép cột

pd.concat(df1, df2, axis = 1)

bind_cols(df1, df2)

11

Lấy các biến là số

df.select_dtypes('float64')

df %>% select_if(is.numeric)

12

Phân rã dữ liệu

pd.melt(df, 'id')

df %>%
pivot_longer(!id)

13

Xoay chiều dữ liệu

df2.pivot_table(index = 'id', columns = 'variable',
values = 'value', aggfunc = 'first').<br> reset_index()

df2 %>%
pivot_wider(id_cods = id, names_from = variable)

1.2.3. Nhóm tổng hợp dữ liệu#

STT

Hàm

Python

R

1

Summary

df.describe()

summary(df)

2

Tổng hợp dữ liệu

Dùng list: df['x'].agg(['mean', 'max'])

df %>% summarise(mean_x = mean(x),
max_x = max(x))

Dùng dictionary: df.agg(['x': ['mean', 'max']])

Dùng tupple: df.agg(mean_x = ('x', 'mean'), max_x = ('x', 'max'))
